# Program-to-Calculate-Power-of-a-Number
# Input two numbers
m = int(input('Enter First value: '))
n = int(input('Enter Second value: '))

# Calculate m raised to the power n
p = m ** n

# Display the result
print('{0} Raise to {1} = {2}'.format(m, n, p))

Enter Second value: 7
